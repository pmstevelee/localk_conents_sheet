# 네이버 카페 크롤러

네이버 카페의 게시글과 댓글을 수집하여 CSV로 저장하는 프로그램입니다.

## 설치 방법

### Windows

1. **Python 설치** (이미 있으면 건너뜀)
   - https://www.python.org/downloads/ 에서 최신 버전 다운로드
   - 설치 시 **"Add Python to PATH"** 반드시 체크

2. **이 폴더를 다운로드** (git clone 또는 ZIP 다운로드)

3. **setup.bat 더블클릭** → 자동 설치 완료

4. **run.bat 더블클릭** → 프로그램 실행

### macOS / Linux

```bash
# 이 폴더로 이동
cd localk_conents_sheet

# 설치 (최초 1회)
bash setup.sh

# 실행
./run.sh
```

---

## 실행 흐름

```
[1단계] 로그인
  1번: 아이디/비밀번호 입력
  2번: 브라우저 쿠키 붙여넣기 (캡차/2FA 발생 시)

[2단계] 카페 이름 입력
  예시) cafe.naver.com/myCafe  →  myCafe  입력

[3단계] 게시판 선택
  번호 입력 (예: 1), 여러 개 (예: 1,3), 전체 (all)

[4단계] 옵션
  - 댓글 크롤링 여부 (Y/n)
  - 최대 페이지 수 (0=전체)
  - 실행 모드
      1 = 지금 1회만 실행
      2 = 매일 자동 실행 (스케줄러)
      3 = 지금 실행 + 매일 자동
```

---

## 출력 파일

`output/` 폴더 안에 날짜별로 저장됩니다.

| 파일 | 내용 |
|------|------|
| `{카페명}_posts_YYYY-MM-DD.csv` | 게시글 목록 (제목, 본문, 작성자, 날짜, 조회수) |
| `{카페명}_comments_YYYY-MM-DD.csv` | 댓글 + 대댓글 (작성자, 내용, 날짜) |

> CSV 파일은 Excel에서 바로 열 수 있습니다 (한글 깨짐 없음).

---

## 캡차 / 2단계 인증 발생 시

1. 브라우저에서 네이버 로그인
2. **F12** → **Application** 탭 → **Cookies** → `cafe.naver.com`
3. `NID_AUT` 와 `NID_SES` 값 복사
4. 프로그램 실행 후 **로그인 방식 2번** 선택
5. `NID_AUT=값; NID_SES=값` 형식으로 붙여넣기

## 주의사항

- 카페 회원이어야 크롤링 가능합니다.
- 너무 빠른 크롤링은 IP 차단을 유발할 수 있습니다 (기본 1.5초 딜레이 설정됨).
- 크롤링한 데이터는 개인적인 용도로만 사용하세요.
